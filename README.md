Active Video Sharing
========================

### Prerequisite
* [NodeJS](http://nodejs.org/download/) > 0.11.9
* [npm](https://www.npmjs.org/)
* [MongoDB](http://www.mongodb.org/downloads)

### Installation
0. ` $ git clone https://github.com/enaqx/videoact `
0. ` $ npm install `
0. ` # npm install -g gulp `
0. ` $ gulp install `
0. ` $ gulp dev `
